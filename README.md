# Rechitect Design

基于MonoRepo形式组织的Vue的企业级模块化工程

编译思想和组件设计方案借鉴了很多[Vben](https://github.com/vbenjs/vben3)的思想

## 基础命令

### 模块生成

```bash
yarn run gen:app <module-name>
yarn run gen:rock <module-name>
```

## IDE的一些注意事项
